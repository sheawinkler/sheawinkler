## Selected Systems

### ContextLattice

Memory and orchestration control plane for agentic systems.

- Grounded context packing across vector, graph, and rollup memory lanes
- Retrieval-control architecture with explicit degradation semantics and traceable fallback behavior
- Multi-runtime control surface for low-latency agent memory operations

Repository: [ContextLattice](https://github.com/sheawinkler/ContextLattice)

---

### Hermes Agent Ultra

Rust-first autonomous runtime focused on policy-safe, long-running execution.

- Deterministic tool policy modes (`audit`, `simulate`, `enforce`) for runtime governance
- Runtime parity discipline with replayable diagnostics and failure forensics
- Structured operator workflow for reliability-by-default autonomous sessions

Repository: [hermes-agent-ultra](https://github.com/sheawinkler/hermes-agent-ultra)

---

### AlgoTrader + FastAPI Sidecar

Execution platform coupling model routing and real-time strategy/risk loops.

- Event-driven trade runtime with bounded risk controls and telemetry-fed adaptation
- FastAPI sidecar as inference/control bridge for Rust execution engines
- Operational focus on execution quality, slippage discipline, and run-context fidelity

Repository: [algotrader-community](https://github.com/sheawinkler/algotrader-community) · [fastapi-sidecar](https://github.com/sheawinkler/fastapi-sidecar)
